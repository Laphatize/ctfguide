Pick a CTF challenge from our extensive database and start a live game. At the end you'll be able to see people's responses and time taken. Perfect for school clubs.

Stuck on a problem? We'll serve you up some Video Tutorials that walk through a similar problem.

If you're a teacher that wants to use CTFGuide to assign homework - we can automatically grade their work and allow easy export to popular grading systems such as Alma and Powerschool.

We have multiple variations of problems stored in our databases. If we detect a problem solution online we automatically delete the problem and replace it with another alternative.
